# ADR 0001: Missing Source Specs at Bootstrap

Date: 2026-02-05
Status: Accepted

## Context
Die Spezifikationsdokumente (`docs/mvp.md`, `docs/roadmap.md`, `docs/architecture.md`, `docs/metrics-checklist.md`) waren beim Projektstart nicht vorhanden.

## Decision
Für Milestone 0 / Issue 0.1 + 0.2 wurde ein minimales Gerüst erstellt und Placeholder-Dokumente angelegt, damit die Struktur, Tooling und das Grundgerüst aufgebaut werden können.

## Consequences
Die Placeholder-Dokumente müssen zeitnah durch die tatsächlichen Spezifikationen ersetzt werden. Weitere Issues sollten danach strikt anhand der finalen Spezifikation umgesetzt werden.
